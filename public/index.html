VisNLP 2.0 | A Visual-Based Educational Support Platform for Teaching and Learning Neural Network-Based NLP Analytics

We propose and devlop a web-based visual platform for learning and teaching nueral-network based NLP analytical methods. Through our work we aim to achieve the following: (1) Present several popular neural network based NLP methods in a digestible format that takes on a linear step-by-step approach. (2) Elimate the 'black box' problem found in nueral network based NLP learning resources through continous real-data examples. (3) Help users interpret model outputs through interactive visual demos that apply nueral-network based NLP method outputs.
